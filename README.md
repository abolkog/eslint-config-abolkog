This is the Linter rules for react native dev based on [eslint-config-aribnbn](https://www.npmjs.com/package/eslint-config-airbnb)

### Usage
1. Install it: 

```npm install --save-dev eslint-config-abolkog```

2. Create .eslintrc file
```
{
    "extends": "abolkog"
}
```
